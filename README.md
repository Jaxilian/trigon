### Dependencies

- [Vulkan SDK](https://www.lunarg.com/vulkan-sdk/)
- [Premake5](https://premake.github.io/)

### Building in a nutshell

#### Windows:
```
cd %USERPROFILE%/Downloads
git clone https://github.com/Jaxilian/trigon.git
cd trigon
premake5 vs2022
tgn.sln
```
#### Linux:
```
cd ~
git clone https://github.com/Jaxilian/trigon.git
cd trigon
premake5 gmak2
make
```
